# Fiszki — польско-русские карточки

Тренажёр польской лексики: ~6000 слов по 43 темам. Один HTML-файл, без зависимостей и бэкенда. Прогресс каждого пользователя хранится в его браузере (localStorage).

## Возможности

- 43 темы + режим «все темы вперемешку»
- Направления PL → RU и RU → PL
- Кнопки «Знаю / Повторить», круг повторения сложных слов
- Прогресс по каждой теме сохраняется между визитами
- Тёмная и светлая тема, управление с клавиатуры (пробел — перевернуть, → знаю, ← повторить)

## Деплой на GitHub Pages

1. Создай новый репозиторий на GitHub (например, `fiszki`).
2. Загрузи в него файл `index.html` (через веб-интерфейс: Add file → Upload files, или через git).
3. Открой **Settings → Pages**.
4. В разделе **Build and deployment** выбери Source: **Deploy from a branch**, Branch: **main**, папка **/ (root)** → Save.
5. Через минуту сайт будет доступен по адресу `https://<твой-ник>.github.io/fiszki/`.

## Как хранится прогресс

Всё лежит в `localStorage` под ключом `fiszki-pl-v1` — у каждого пользователя свой прогресс в своём браузере. Кнопка «Сброс» в шапке очищает его. Сервер и база данных не нужны.
